Inf

Inf is a calculator that can handle infinite and infinitesimal numbers. This is an early alpha release, and it's more likely to work well in Firefox than in Internet Explorer. Type an expression at the > prompt and hit Enter. For more documentation, scroll down.

About Inf

Inf is a calculator that can handle infinite and infinitesimal numbers.

Predefined constants:

  • i -- the square root of -1
  • pi -- the ratio of a circle's circumference to its diameter
  • d -- an infinitesimally small number

Built-in operators:

  • : -- assigns the expression on the right to the variable named on the left-hand side
  • =, <, > -- comparisons
  • +, -, *, / -- arithmetic
  • ^ -- exponentiation
  • Statements can be separated by semicolons.

Built-in functions:

  • sin, cos, tan
  • asin, acos, atan
  • sqrt, abs
  • exp, ln
  • floor, ceil

User-defined functions:

  • f x: x^2

Inf represents infinite and infinitesimal quantities as Levi-Civita numbers. The Levi-Civita numbers obey all the same elementary axioms of arithmetic as the real numbers. If you try evaluating the expression 1/0, you'll see that it comes up as undefined, not an infinite number. This is because one of the elementary axioms of arithmetic is that division by zero is undefined. To get an infinite result, you could do something like 1/d, where d is the symbol built in to the calculator to represent one particular infinitesimal number. Another thing that can be proved from the elementary axioms of arithmetic is that if x is greater than zero, then 2x is greater than x. This applies to Levi-Civita numbers as well, so infinite and infinitesimal numbers have to come in different sizes. In particular, there isn't anything in the Levi-Civita system that plays the role of ∞, which is usually used as a generic symbol for infinity.

Here are some examples you can try in the calculator to get a feeling for how all this works:

  • 1/0 ... undefined
  • 1/d ... gives an infinite result
  • 2/d ... a bigger infinity
  • 2/d+1 ... and an even bigger infinity
  • d>0 ... true
  • d<10^-10 ... true
  • d^2<d ... true, showing that we have different sizes of infinity
  • d^2>0 ... true, because even though d^2 is infinitesimally small compared to d, it's still no zero
  • sqrt(d)
  • sqrt(d)>d ... true
  • 1/(1-d) ... is calculated using the geometric series, up to a certain level of precision
  • exp(d) ... is calculated using a Taylor series
  • exp(1/d) ... is undefined in the Levi-Civita system

Calculus students often get the impression that the notion of an infinite or infinitesimal number can never be defined in any rigorous or self-consistent way. That's not true. Tullio Levi-Civita defined the Levi-Civita numbers around 1900, so it's been known for over a century that there were rigorously definable number systems that included infinities. Abraham Robinson defined another such number system, the hyperreal numbers, around 1960. The hyperreals match the properties of the real numbers even better than the Levi-Civita numbers do. For example, the exponential function can take a positive infinite argument in the hyperreal system, but that doesn't work in the Levi-Civita numbers. However, the hyperreals cannot be conveniently represented on a computer. Great mathematicians like Newton, Gauss, and Euler used infinitesimals, and now that modern mathematicians like Levi-Civita and Robinson have put infinitesimals on a more rigorous footing, there's no reason to shy away from them.
